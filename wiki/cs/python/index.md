# 파이썬 프로그래밍 스터디 대시보드 (Python Programming Study Dashboard)

이 문서는 파이썬 프로그래밍 교재의 전반적인 개념을 깊이 있게 정리한 스터디 노트의 통합 진입점입니다.

---

## 학습 로드맵 및 바로가기

- **[Chapter 02: 파이썬 개론 및 개발환경](chapter-02.md)** - 컴파일러와 인터프리터의 동작 차이, CPython 번역 모델 및 PVM 실행 원리.
- **[Chapter 03: 데이터와 변수](chapter-03.md)** - 변수의 메모리 매핑 기법, 6대 명명 규칙, 자료형 변환 및 정밀도 유실 예외.
- **[Chapter 04: 연산자](chapter-04.md)** - 논리 연산의 단락 평가(Short-circuit Evaluation), 삼항 조건식 및 operator 모듈 최적화.
- **[Chapter 05: 조건문](chapter-05.md)** - if 분기 구조와 들여쓰기 규칙, pass 키워드의 물리적 역할 및 다중 비교 순서 설계 주의사항.
- **[Chapter 06: 반복문](chapter-06.md)** - for문과 range() 증감 규칙, while문 제어 변수 관리, continue/break 및 중첩 루프 실습.
- **[Chapter 07: 리스트](chapter-07.md)** - 컨테이너 리스트의 물리적 메모리 참조 방식, 주요 삽입/삭제 메서드 작동 원리 및 슬라이싱.
- **[Chapter 08: 튜플과 딕셔너리](chapter-08.md)** - 튜플의 불변성 규칙과 리스트 형변환, 해시 테이블 기반 딕셔너리의 Key 불변 제약 및 조회/수정 연산.
- **[Chapter 09: 함수 기초](chapter-09.md)** - 사용자 정의 함수의 문법 구조, PEP 8 스타일 가이드 표준, 코드 재사용성과 모듈화의 장점.
- **[Chapter 10: 함수 활용](chapter-10.md)** - 지역/전역 변수 스코프와 global 키워드, 가변인자(*args) 및 다중 리턴 튜플, 중첩/재귀 함수 제어.
- **[Chapter 11: 모듈](chapter-11.md)** - 사용자 정의 모듈 로드와 sys.path 확장, 메인 가드 블록(`__name__ == '__main__'`) 구축 및 표준 모듈(math, random, time) 분석.
- **[Chapter 12: 객체지향 프로그래밍](chapter-12.md)** - OOP의 속성/메서드 지향 철학, 클래스와 인스턴스의 논리적 관계, self 참조 및 `__init__` 생성자의 역할.
- **[Chapter 13: 파일 입출력](chapter-13.md)** - 파일 처리 3단계 라이프사이클, 절대/상대 경로 및 raw string 지정, cp949 디코딩 에러 방지 및 write 캐스팅 규칙.
- **[Chapter 14: 예외처리](chapter-14.md)** - 구문/논리/실행시 에러의 차이점, try-except-else-finally 통합 블록 및 상황별 3대 예외 제어 흐름 분석.
- **[Chapter 15: 람다함수와 리스트축약](chapter-15.md)** - 익명 함수 람다의 장단점, filter/map의 지연 평가(Lazy Evaluation) 메커니즘, 리스트 축약식의 다중 필터링 및 join() 문자열 병합.
