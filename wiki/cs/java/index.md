# Java Programming

이 섹션은 자바 프로그래밍과 관련된 개념, 에러 노트, 모범 사례 등을 정리하는 공간입니다.

## Study Notes

- [Chapter 01: Development Environment Setup (개발 환경 구축)](chapter-01.md) - 자바 응용 개발 환경(JDK, JRE, JVM) 구축 및 이클립스 개발 프로세스
- [Chapter 02: Java Basic Programming - Basics & Operators & Conditionals (자바 기본 구조, 연산자 및 조건문)](chapter-02.md) - 자바 기본 구조, 데이터 타입(기본/참조), 타입 변환(캐스팅), Scanner 키 입력, 연산자 및 조건문
- [Chapter 03: Control Flow - Loop Statements (제어 흐름 - 반복문)](chapter-03.md) - 자바 반복문(for, while, do-while) 작동 원리, 중첩 반복, 루프 제어(break, continue) 및 문자열 equals 비교
- [Chapter 04: Data Structures & Diagnostics - Arrays & Exception Handling (배열 및 예외 처리)](chapter-04.md) - 1D/2D/비정방형 배열 힙 영역 메모리 참조 원리, length 필드, 예외 처리(try-catch-finally) 및 예외 클래스별 대응
- [Chapter 05: Modular Programming - Methods (모듈식 프로그래밍 - 메서드)](chapter-05.md) - 메서드 정의 및 구조, static 지정자, 4가지 입출력 패턴 및 연습 예제 분석
- [Chapter 06: Class and Object (클래스와 객체)](chapter-06.md) - 자바의 객체지향 프로그래밍(OOP) 기본 개념인 클래스와 객체, 그리고 캡슐화, 상속, 다형성, 추상화의 주요 개념과 자바에서의 클래스 정의, 생성자, 접근 제한자의 기초를 다룹니다.
- [Chapter 07: Inheritance (상속)](chapter-07.md) - 자바의 핵심 객체지향 개념인 상속(Inheritance)의 특징, 선언 방법, 상속과 접근 지정자 간의 관계, 생성자 호출 순서 및 super()의 활용법, 그리고 메서드 오버라이딩과 final 예약어의 사용법을 다룹니다.
- [Chapter 08: Extension of Inheritance (상속의 확장)](chapter-08.md) - 상속 개념의 확장인 업캐스팅과 다운캐스팅, instanceof 연산자, 동적 바인딩의 메커니즘과 다형성의 조건, 그리고 추상 클래스와 인터페이스의 비교 분석을 심도 있게 다룹니다.
- [Chapter 09: Module and Package (모듈과 패키지)](chapter-09.md) - 자바의 네임스페이스 및 배포 구조인 패키지와 모듈 시스템을 살펴보고, 자바 핵심 내장 라이브러리 클래스인 Object, Wrapper, String, StringBuffer, StringBuilder, Math, Calendar, 그리고 java.time 패키지의 상세 구현과 특징을 분석합니다.
- [Chapter 10: GUI Programming Basics (GUI 프로그래밍 기초)](chapter-10.md) - 자바 GUI 프로그래밍의 기초인 AWT와 Swing 패키지의 차이점을 분석하고, 다양한 배치 관리자(Layout Manager)의 사용법 및 이벤트 처리(Event Handling) 모델인 리스너, 어댑터, 마우스/키 이벤트 처리 방식에 대해 심도 있게 다룹니다.
- [Chapter 11: Swing Components I (스윙 컴포넌트 1)](chapter-11.md) - 스윙 컴포넌트들의 최상위 공통 속성을 제공하는 JComponent 클래스의 주요 메서드를 살펴보고, JLabel, JButton, JCheckBox, JRadioButton, JTextField, JTextArea, JList, JComboBox, JSlider 등 핵심 컴포넌트들의 생성과 이벤트 제어 방식을 상세히 학습합니다.
- [Chapter 12: Swing Components II - Part A (스윙 컴포넌트 2 - Part A)](chapter-12.md) - 스윙의 핵심 표 형태 컴포넌트인 JTable과 DefaultTableModel의 동작 메커니즘을 상세 분석하고, 메뉴바(JMenuBar), 메뉴(JMenu), 메뉴 아이템(JMenuItem) 등 메뉴 트리 구축 기법 및 도구 모음인 JToolBar의 제어 방법을 학습합니다.
- [Chapter 13: Swing Components II - Part B (스윙 컴포넌트 2 - Part B)](chapter-13.md) - 스윙 애플리케이션의 팝업 창 및 보조 윈도우 제어를 담당하는 JDialog의 모달/모달리스 모드를 비교하고, JOptionPane을 통한 유형별 팝업 메시지 박스 제어, JFileChooser 및 JColorChooser를 이용한 로컬 자원 접근법을 살펴봅니다.
- [Chapter 14: Java File Input/Output (자바 파일 입출력)](chapter-14.md) - 자바의 입출력 스트림 개념과 바이트/문자 스트림, 버퍼 스트림(BufferedReader/BufferedWriter) 활용법
- [Chapter 15: Java Database Connectivity (JDBC)](chapter-15.md) - 자바 RDBMS 연동 표준 기술인 JDBC 아키텍처 및 핵심 API(Connection, Statement, ResultSet) 분석
- [Chapter 16: Model-View-Controller (MVC Pattern)](chapter-16.md) - 애플리케이션 아키텍처 디자인 패턴인 MVC 패턴의 정의, 구성 요소 역할 및 데이터 흐름 분석
- [Quiz: Java Practical Programming Exercises (자바 실무 프로그래밍 퀴즈 풀이)](quiz.md) - 자바 프로그래밍 기초 논리, 예외 처리, 컬렉션 프레임워크 및 정렬 알고리즘 퀴즈 풀이
